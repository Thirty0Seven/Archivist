# Archivist
A simple tool that Packages directories into self-extracting shell scripts.
That's all there realy is to it.
When run it uses zenity for folder selection and outputs a .sh in the same directory as the source feel free to tinker i just made it to simplify the act of making these kinds of archives.

In order to install it yu need to have zenity and unzip
