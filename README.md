# Archiver
A simple tool that Packages directories into self-extracting shell scripts.
That's all there realy is to it its only meant to be used on the Steam Deck as it relies on a recreation of the directory structure of SteamOS (/home/deck/...).
when run it uses zenity for folder selection and outputs a .sh i the same directory as the source feel free to tinker i just made it to simplify the act of making these kinds of archives.
